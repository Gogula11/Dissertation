# Glossary and Abbreviations

| Term | Definition |
|------|------------|
| **A2C** | Advantage Actor-Critic — a synchronous variant of the A3C policy gradient algorithm |
| **Composite score** | Normalised weighted sum of weighted tardiness and setup cost objectives |
| **DEAP** | Distributed Evolutionary Algorithms in Python — a GA framework used for the base solver |
| **DQN** | Deep Q-Network — a value-based DRL algorithm using neural network function approximation |
| **DRL** | Deep Reinforcement Learning — RL with deep neural network function approximation |
| **GA** | Genetic Algorithm — an evolutionary optimisation method inspired by natural selection |
| **Giant-tour encoding** | A chromosome representation where a single permutation of all jobs is split into machine segments |
| **Gymnasium** | An API standard for RL environments (successor to OpenAI Gym) |
| **Hyper-heuristic** | An automated methodology for selecting or generating heuristics to solve search problems |
| **Mutation operator** | A GA operator that introduces random variation into offspring chromosomes |
| **NN-Greedy** | Nearest-Neighbour Greedy — a constructive heuristic accounting for setup costs |
| **OX** | Order Crossover — a GA crossover operator that preserves relative job ordering |
| **PMSP** | Parallel Machine Scheduling Problem — assigning n jobs to m identical parallel machines |
| **PMSP-SDSC** | PMSP with asymmetric Sequence-Dependent Setup Costs |
| **PPO** | Proximal Policy Optimisation — a policy gradient DRL algorithm with clipped surrogate objective |
| **SAC** | Soft Actor-Critic — a maximum-entropy DRL algorithm |
| **SB3** | Stable-Baselines3 — a library of reliable DRL implementations |
| **SDSC** | Sequence-Dependent Setup Costs — costs that depend on the specific pair of consecutive jobs |
| **SPT** | Shortest Processing Time — a heuristic that sorts jobs by ascending processing time |
| **Wilcoxon signed-rank test** | A non-parametric paired difference statistical test |
